Commercial Expression

Kitware is teaming with Simmetrix, Inc. http://www.simmetrix.com to create commericial tools that integrate the VTK visualization environment to Simmetrix's products, and to integrate Simmetrix's integration framework into ParaView.

If you are interested in building an adapted framework to VTK from your analysis/simulation package, please email sales "at" kitware.com for more information.
